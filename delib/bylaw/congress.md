# 公司代表大会议事章程

## 一、基本原则

1. **会前准备**：书记处须提前整理会议材料，确保大会能形成有效决议
2. **问题导向**：各委员会从大会实际问题出发，讨论形成提案
3. **层级分明**：各议事机构独立运作，重大事项提交代表大会决策
4. **新人培养**：通过轮值机制，让新成员通过书记处了解全局

## 二、组织架构

```
公司代表大会（周会）
    ├── 执行委员会（管理会）  ← 秘书处暂代
    ├── 技术委员会（技术会）  ← 秘书处暂代
    └── 书记处（办公会）
```

| 机构 | 会议名称 | 频率 | 当前状态 |
|------|----------|------|----------|
| 公司代表大会 | 周会 | 每周一次 | 最终决策 |
| 执行委员会 | 管理会 | 每周一次 | 秘书处暂代 |
| 技术委员会 | 技术会 | 每周一次 | 秘书处暂代 |
| 书记处 | 办公会 | 每周一次 | 正常运行 |

各议事机构自行组织自己的每周会议。

---

## 三、职责与运作

### 1. 公司代表大会（周会）

- **决策方式**：需全体与会人员明确表决通过
- **材料来源**：书记处汇总整理各委员会提案
- **效率保障**：书记处未提前准备时，会议难以形成有效决议

### 2. 执行委员会（管理会）

- 由秘书长主持，部门负责人参与
- 每周精选 1-2 件核心管理事项，深入分析原因
- 形成提案后提交至周会
- 需要更多部门负责人参与时，可扩大为执行委员会正式会议

### 3. 技术委员会（技术会）

- 由 CTO助理主持（与 CEO 共同召开），技术负责人参与
- 讨论技术事务，形成提案
- 需要更多技术负责人参与时，可扩大为技术委员会正式会议

### 4. 书记处（办公会）

- 由轮值书记负责统筹
- 每周定时审议各会议内容，了解全局
- 提前整理材料提交至代表大会
- 负责新人轮值培养

---

## 四、运作流程

```
发现问题 → 对应委员会会议讨论 → 形成提案 → 书记处汇总整理 → 代表大会决策 → 循环改进
```

**示例**：浙理工款项未到导致现金流问题 → 管理会专题分析原因与整改 → 形成提案 → 书记处整理 → 周会决策

---

## 五、新人培养机制

- **轮值书记参与**：轮值书记参与书记处会议
- **培养对象**：优先安排有培养空间的人员
- **职责**：负责根据讨论结论整理会议材料
- **SSC 成员**：侧重参与书记处工作，借此了解全局
