# 新媒体运营笔试题

考核项目：量潮课堂 · 新媒体运营
考核时长：2-3天
提交物：改写稿 + 思路说明 + 简要规划
工作环境：远程异步

## 笔试任务

### 任务一：分析一篇高转化内容

阅读《重构是 Vibe Coding 时代的必备技能》（文章来源：微信公众号「量潮科技」），回答：

1. 列出文章结构大纲，标注每段作用。
2. 哪几个点推动了读者的信任和咨询意愿？为什么有效？
3. 这篇文章克制了什么没写？为什么这种克制有效？
4. 提炼 3 条可复用的内容创作原则。

### 任务二：演绎一段公众号开头

阅读《如何快速向新人成套传递十年老手的工作体系？》（文章来源：微信公众号「量潮科技」）。原文是一篇故事。请根据这个故事演绎成一段公众号开头。

要求：
- 读者读完产生两个感受：「这说的就是我的问题」「我想知道他们怎么做到的」
- 不要写成课程广告

提交时附上思路说明：原文哪些信息点启发了你的开头？你做取舍的依据是什么？

## 提交要求

一份 Markdown 文档，包含：任务一（文章拆解 + 创作原则）、任务二（改写稿 + 思路说明）。

评估维度：改写能力、叙事意识、用户视角。
