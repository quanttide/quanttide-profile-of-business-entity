# 项目经理考核题

## 笔试

### 考核题

我们承接企业数字化转型项目，涉及需求分析、进度管理、质量控制和客户沟通。项目经理负责从签约到交付的全流程管理。

请阅读微信公众号「量潮科技」的文章，了解我们的业务风格和交付标准。然后实际找一件需要协作完成的事情（可以是帮朋友组织一次活动、协调一个小项目、或推进一件需要多方配合的事），负责把它推进到完成。

记录你的项目记录：
1. 你做了什么事？涉及哪些人？
2. 你是怎么拆解任务和分工的？
3. 推进过程中遇到了什么问题？你怎么协调解决的？
4. 最终结果怎么样？如果顺利，什么起了作用？如果不顺利，问题出在哪？

回复格式：一份项目记录，包含你的计划和复盘分析。
