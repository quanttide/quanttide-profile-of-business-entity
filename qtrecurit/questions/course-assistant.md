# 课程助教考核题

## 笔试

### 考核题

量潮课堂提供 AI 时代的编码体系课程，包含实践课和实训基地。助教负责管理学员的学习进度、组织实训、收集反馈。

请阅读微信公众号「量潮科技」的文章，了解课堂服务的内容和风格。然后实际教一位朋友或熟人 Vibe Coding（内容自选，可以是编程、工具使用或其他技能），或带他完成一次实训任务。

记录你的教学记录：
1. 你教了什么？为什么选这个内容？
2. 教学过程中对方遇到了什么问题？你怎么引导的？
3. 对方学到了什么？你怎么判断他学会了？
4. 如果教学效果不理想，你觉得原因是什么？如果效果好，什么起了作用？

回复格式：一份教学记录，包含你的教学准备和过程复盘。
